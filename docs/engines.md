# Conversion Engines

## Pandoc (recommended)

- Best fidelity for complex documents
- Supports tracked changes, Lua filters, footnotes
- Requires separate installation: https://pandoc.org/installing.html

## Mammoth

- Pure Node.js, no external binary required
- Good for basic documents; converts via HTML intermediate
- Supports custom style mappings

## LibreOffice

- Broadest format support
- Requires LibreOffice installation
- Markdown export quality varies by version
