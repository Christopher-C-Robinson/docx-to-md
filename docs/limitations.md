# Known Limitations

- **Complex tables**: Tables with merged cells fall back to raw HTML in GFM output.
- **Embedded fonts**: Font information is not preserved in Markdown.
- **SmartArt / DrawingML**: Rendered as placeholder text or omitted.
- **Macros**: VBA macros are ignored.
- **Password-protected files**: Not supported; decrypt before converting.
- **LibreOffice Markdown export**: Quality depends on LibreOffice version; may produce plain text.
